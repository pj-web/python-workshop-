Заголовки в HTML

Заголовки используются для структурирования и организации контента на веб-странице. Они имеют разные уровни важности, начиная от h1 (наиважнейший) до h6 (наименее важный). Использование каскада заголовков помогает сделать вашу страницу более доступной для интерпретации браузерами и инструментами скринридеров.

Тег <p> - текст в HTML

Параграфы p служат для обертки текста в блоки. Они обычно создают отступы сверху и снизу, разделяя текст из разных параграфов друг от друга. Это помогает в форматировании текста и удобочитаемости содержимого.

Ссылки <a> 😎

Внешние ссылки

w3schools

Тег a используется для создания гиперссылок на страницах. Он может содержать внутри ссылку на другую страницу, адрес электронной почты, файл или другой тип ресурса. Атрибут href указывает URL-адрес целевого ресурса. Ссылка может быть внешней (на другой сайт) или внутренней (в пределах вашего сайта).

Изображения <img>

  • img - используется для вставки изображений на страницу.
  • Можно использовать атрибут src, чтобы указать URL-адрес изображения.
  • Также используйте атрибут alt для предоставления описания изображения для скринридеров или случаев, когда изображение не может быть загружено.
  • width - ширина изображения
  • height - высота изображения

Таблицы <table>

Для создания таблиц используются следующие теги:

  • table: основной контейнер таблицы
  • tr: строка таблицы (table row)
  • th: ячейка-заголовок (table header)
  • td: ячейка таблицы (table data)

Семантические теги

Семантические теги в HTML предоставляют информацию о значении содержимого между открывающим и закрывающим тегами. В отличие от структурных тегов, таких как div или span, которые не передают никакой конкретной информации о содержимом внутри них, семантические теги помогают определить, какую роль играет определенный участок контента на веб-странице.

  • header - Заголовок или верхняя часть сайта/раздела.
  • nav - Навигационное меню.
  • main - Основное содержимое веб-страницы. Должен быть только один на странице.
  • article - Самостоятельный, независимый контент (например, блоговая запись).
  • section - Раздел внутри документа (например, глава, вкладка, вкладка на виджете и т. д.).
  • aside - Содержимое, отделенное от основного содержимого, но связанное с ним (например, боковая панель или врезка).
  • footer - Нижний колонтитул сайта/раздела.
  • time - Время или дата.
  • figure и figcaption - Изображение или иллюстрация и ее подпись.
  • mark - Выделенный текст.
  • code - Код программирования.
  • pre - Предварительно отформатированный текст.
  • blockquote - Цитата.
  • q - Короткая цитата.
  • abbr - Аббревиатура.
  • address - Адрес.
  • cite - Ссылка на источник.
